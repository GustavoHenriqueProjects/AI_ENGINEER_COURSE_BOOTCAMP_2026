Criação do ambiente virtual: conda create --name chatbot_pdf python=3.11

Ativação do ambiente virtual: conda activate chatbot_pdf

Para o desenvolvimento visual do chat: pip install streamlit

Para rodar a aplicação usamos: streamlit run app.py

Para usar o gerenciador de modelos OLLAMA use: python -m spacy download pt_core_web_sm

Para vircular ao ambiente python vamos usar : python -m ipykernel install --user --name=npl_course_env
